# Calc - standard calculator
A standard calculator featuring addition, subtraction, multiplication, and division. It also includes an option to either exit or continue performing operations. Just like a standard calculator, there is nothing unusual here

  ## How to Run 
 1. Clone the repository:
   ```
   bash
   git clone https://github.com/nam313ss/Calc---standard-calculator---
   ```

  2. Navigate
 ```  cd your-repo-name  ```

  3. Run the application
     ``` dotnet run ```


     ## Features
- Accepts two numbers and an operator (+, -, *, /).
- Handles division by zero with an explicit error message.
- Validates numeric input (catches `FormatException`).
- Allows repeated calculations in a loop; exits on 'n'.
